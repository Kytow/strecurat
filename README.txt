Utilisation du logiciel :
Lancez modules.bat, cela permet d'installer les modules python nécessaires.
Lancez strecurat.bat
Choisissez si vous voulez faire une recherche EU ou US.
Choississez le nombre de fenêtres à ouvrir (Le programme est très optimisé, 10 fenêtres suffisent largement pour une utilisation normale)
Laissez le programme agir, le nombre de fenêtres que vous avez choisies vont se lancer
Tous les serveurs Ngrok trouvés apparaissent dans server-list.txt avec le nombre de joueurs connectés, le ping et la version du serveur.
